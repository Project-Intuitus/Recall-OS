## What's New in v1.0.4

Quality-of-life fixes and payment provider migration.

### Fixes

**Hidden FFmpeg Console Window**
- FFmpeg no longer opens a visible command prompt window during video/audio extraction
- All media processing now runs silently in the background

### Changes

**Payment Provider Migration**
- Migrated from LemonSqueezy to Paddle as merchant of record
- License purchasing still coming soon — Paddle account under review
- Join the waitlist at projectintuitus.com to be notified when licensing launches

### Installation

1. Download `RECALL.OS_1.0.4_x64-setup.exe` below
2. Run the installer
3. Get your API key from [Google AI Studio](https://aistudio.google.com/apikey)
4. Paste in Settings → Validate → Start indexing

### Requirements

- Windows 10/11 (x64)
- 4GB RAM minimum
- Gemini API key (free tier available)
